lamp环境搭建（三）

PHP篇

安装centos6.5自带php

安装相关模块：为了让PHP支持MySQL，我们可以安装php-mysql软件包；也可使用以下命令搜索可用的php模块
[root@localhost ~]# yum -y install php php-mysql

安装php常用扩展

其中gd-devel php-mbstring php-imap找不到镜像，要下载rpm包才可以，但是找不到php-imap-5.3.3-26-el6-x86_64的包，所以没装

重启httpd服务，这一步很重要

[root@localhost ~]# service httpd restart

然后，我们提供php页面，测试
